## Simulacion de Supermercado con Multi Agentes

Para ejecutar este proyecto solo descarga o clona este repositorio y ejecuta el archivo `main.py` con el comando:
>`python main.py` 

desde el terminal o desde tu editor de codigo favorito.

P.D. Recuerda tener instalado `matplotlib`! Si no lo tienes instalado escribe:
>`pip install matplotlib` 

en el mismo directorio donde tienes descargado este repositorio.
